## Installation

You only need to install the gem for your specific bank. The main `bankscrap` gem will be installed automatically as a dependency.

### Using RubyGems

```bash
# Example: BBVA
gem install bankscrap-bbva

# Example: ING
gem install bankscrap-ing
```

### Using Bundler

Add to your `Gemfile`:

```ruby
# Example: BBVA
gem 'bankscrap-bbva'

# Example: ING
gem 'bankscrap-ing'
```

## Request a New Bank

Interested in adding support for another bank? [Open an issue](https://github.com/bankscrap/bankscrap/issues) on the main repository and the community will try to help.

You can also contribute by [creating your own adapter](/adapters/generator) using the bankscrap generator.
